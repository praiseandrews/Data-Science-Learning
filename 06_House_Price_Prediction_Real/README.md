# House Price Prediction (Real Dataset)

## Overview

A real-world machine learning project using a housing dataset from Kaggle.

## Dataset

Source:
Housing Prices Dataset from Kaggle

Download the dataset from:
https://www.kaggle.com/datasets/yasserh/housing-prices-dataset

## Skills Learned

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- One-Hot Encoding
- Linear Regression
- Model Evaluation
- Model Saving

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Outputs

- Correlation Heatmap
- Price Distribution Plot
- Actual vs Predicted Plot
- Trained Model (.pkl)

## Learning Outcome

This project demonstrates a complete machine learning workflow from raw data to model deployment readiness.
