# House Price Prediction (Toy Project)

## Overview

This is my first Machine Learning project.

The goal is to train a model that can predict the price of a house based on its area.

## Dataset

A small sample dataset created for learning purposes.

| Area (sq ft) | Price |
|-------------|--------|
| 1000 | 50000 |
| 1200 | 60000 |
| 1500 | 75000 |
| 1800 | 90000 |
| 2000 | 100000 |

## Machine Learning Workflow

1. Load Data
2. Select Features
3. Split Data into Training and Testing Sets
4. Train Linear Regression Model
5. Make Predictions
6. Evaluate Results

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Google Colab
